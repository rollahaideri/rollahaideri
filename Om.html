Vad är Css?


CSS står för Cascading Style Sheets och är en av webbens tre grundläggande byggstenar tillsammans med HTML och Javascript.

Man kan säga att CSS är webbens ansikte utåt, och sköter alltså allt som har med design att göra.

Med hjälp av CSS kan man åstadkomma allt från att välja typsnitt på text till att göra snygga animationer.
